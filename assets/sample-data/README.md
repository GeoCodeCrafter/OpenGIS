# Sample Data

This directory will contain sample georeferencing test data.

## Included References

OpenGIS bundles references to these free datasets via the catalog service:

### Tile Basemaps
- **OpenStreetMap** — General-purpose world map
- **CARTO Light** — Light base for data overlay
- **CARTO Dark** — Dark base for data overlay
- **CARTO Voyager** — Detailed, labeled world map
- **Stadia Stamen Toner** — High-contrast black & white

### Vector Data (Natural Earth)
- **Natural Earth Countries** — Country boundaries (1:110m)
- **Natural Earth Rivers** — Major rivers (1:110m)
- **Natural Earth Cities** — Populated places (1:110m)

Data sourced from [Natural Earth](https://www.naturalearthdata.com/) (public domain).

## Testing Images

For testing the georeferencing workflow, you can use:

1. **USGS Topographic Maps** — Available from [The National Map](https://apps.nationalmap.gov/downloader/)
2. **David Rumsey Map Collection** — Historical maps at [davidrumsey.com](https://www.davidrumsey.com/)
3. **Library of Congress** — Historical maps at [loc.gov/maps](https://www.loc.gov/maps/)

These sources provide non-georeferenced scanned maps with visible coordinate labels, ideal for testing automatic detection.
